Ayo Mengenal

HTML, CSS, JavaScript & Tailwind.

HTML

Kerangka atau struktur dasar dari sebuah halaman web. Ibaratnya, ini adalah tulang-belulang website.

CSS

Digunakan untuk mendesain tampilan website, seperti warna, layout, dan font. Ini adalah "kulit" dan "pakaian" dari website.

JavaScript

Bahasa yang membuat website menjadi interaktif dan dinamis. Ini adalah "otak" dan "otot" website.

Tailwind CSS

Framework CSS yang mempercepat proses styling dengan menyediakan class-class siap pakai (utility-first).


Langkah Membuat Website

1

Persiapan

Siapkan Text Editor (seperti VS Code) dan Web Browser.

2

Struktur (HTML)

Buat file index.html dan susun kerangka konten dengan tag.

3

Gaya (CSS)

Buat file style.css untuk mendesain warna, font, dan layout.

4

Interaksi (JS)

Buat file script.js untuk menambahkan animasi dan fungsi interaktif.

5

Publikasi

Unggah folder proyek ke layanan gratis seperti Vercel atau GitHub Pages.
